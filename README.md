# Personal Portfolio Website

A responsive and interactive personal portfolio website built using HTML, CSS, and JavaScript. This project showcases my profile, technical skills, projects, and contact details in a clean and user-friendly interface.

## Features

- Responsive design for desktop and mobile devices
- Sticky navigation bar
- Dark mode toggle
- Animated typing effect
- Smooth scrolling navigation
- Scroll reveal animations
- Skills section with Font Awesome icons
- Projects section with GitHub links
- Contact form with JavaScript validation
- Clean and modern UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Google Fonts (Poppins)

## How to Run

1. Clone this repository

```bash
git clone https://github.com/mandal-rupa/portfolio-website.git
```

2. Open the project folder.

3. Open `index.html` in any modern web browser.
